Селектори CSS - це спосіб вказати CSS-стилі для певних елементів HTML. Є багато різних типів селекторів, але деякі з найпоширеніших включають: Селектор за назвою елемента обирає всі елементи з певною назвою, наприклад div або p. Селектор за id обирає елемент з певним ідентифікатором, наприклад #main. Селектор за класом обирає всі елементи з певним класом, наприклад .red або .green. Селектор за кількома класами обирає всі елементи з певною комбінацією класів, наприклад .red.bold або .green.italic. Селектор за вкладеністю обирає елементи, які є нащадками певного елемента, наприклад div > p або ul li. Селектор за сестринськими елементами обирає елементи, які є сестринськими елементами певного елемента, наприклад div + div або ul ~ li. Селектор за псевдо-класом обирає елементи на основі їхніх станів або поведінки, наприклад :hover або :nth-child(2).
